INLINE CSS :-

Inline CSS is applied directly to HTML elements using the style attribute. It allows you to apply CSS styles to individual HTML tags. Inline CSS is the technique to define the single element with the insert style sheets in an HTML document. We can add CSS in three approaches: Inline, Internal, and External. It has the interactive and unique style to create a single HTML element; we can define the inline CSS on the style attribute.

Note- Inline CSS is not reusable and can't be use for other tags even in the same document, It also increases HTML file size, hard to maintain, breaks the principle of separation of concerns.
